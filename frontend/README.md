# KAMEXCHANGE Frontend

Digital Asset Exchange Platform Frontend built with Next.js 15

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:8081](http://localhost:8081) with your browser.

## Features

- Next.js 15 with App Router
- TypeScript
- TailwindCSS
- RTL Support (Persian/Arabic)
- i18n (Persian/English)
- TradingView Lightweight Charts
- Dark/Light Theme
